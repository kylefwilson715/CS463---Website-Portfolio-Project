# Portfolio Website

Personal portfolio website showcasing my computer science experience, projects, and coursework.

## Deployed Site

Live site: https://kylefwilson715.github.io/CS463---Website-Portfolio-Project/

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.0

## File Structure

```
CS463---Website-Portfolio-Project/
├── index.html           # Home page
├── about.html           # About page
├── experience.html      # Experience and skills
├── projects.html        # Project showcase
├── contact.html         # Contact information and form
├── css/
│   └── styles.css       # Custom styles
├── js/
    └── main.js          # JavaScript functionality
```

## How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/kylefwilson715/CS463---Website-Portfolio-Project.git
```

2. Navigate to the project directory:
```bash
cd CS463---Website-Portfolio-Project
```

3. Open `index.html` in your web browser
